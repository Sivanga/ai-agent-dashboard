# Returns Policy

## Standard Returns
- Customers can return any item within 30 days of purchase
- Items must be in original packaging and unused condition
- Refunds are processed within 5-7 business days
- Return shipping is free for defective items
- Non-defective returns have a Â£5.99 return shipping fee

## Exceptions
- Sale items are final sale and cannot be returned
- Personalized/custom items cannot be returned
- Electronics must be returned within 14 days
- Gift cards are non-refundable

## How to Start a Return
1. Log into your account at myaccount.example.com
2. Go to Order History
3. Click "Return Item" next to the product
4. Print the prepaid return label
5. Drop off at any Royal Mail location

## Exchanges
- We offer free exchanges for different sizes/colors
- Exchange items ship within 2-3 business days
- If the new item costs more, you'll be charged the difference
